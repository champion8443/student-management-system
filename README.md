\# Student Management System



A simple full-stack web application to manage student records (CRUD operations).



\## Technologies Used

\- \*\*Frontend\*\*: HTML, CSS, JavaScript

\- \*\*Backend\*\*: Java (Servlet + JSP + JDBC)

\- \*\*Database\*\*: MySQL

\- \*\*Build/IDE\*\*: IntelliJ IDEA, JDK 22

\- \*\*Version Control\*\*: Git + GitHub



\## Features (Planned)

\- Add new student

\- View all students

\- Update student details

\- Delete student

\- Basic form validation

\- Clean UI with CSS
